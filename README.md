# AI-Powered YouTube Quiz Generator

An intelligent quiz generation system that automatically creates multiple-choice questions (MCQs) from YouTube videos using Retrieval-Augmented Generation (RAG).

The application extracts video transcripts, performs semantic chunk retrieval using Sentence Transformers and FAISS Vector Search, and leverages Groq's Llama models to generate high-quality quizzes. Users can take interactive quizzes, receive instant scoring, and review correct answers.

##  Features

- Extract transcripts from YouTube videos
- Intelligent transcript chunking
- Semantic retrieval using FAISS
- Embedding generation using Sentence Transformers
- AI-powered MCQ generation using Groq LLM
- Interactive Streamlit interface
- Automatic score calculation
- Detailed quiz review and answer analysis
- Retrieval-Augmented Generation (RAG) architecture
